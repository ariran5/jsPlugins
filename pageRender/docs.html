Инициализация

Первой строчкой кода будет запоминание в переменную объекта

	var a = new pageDraw();

Можно добавить параметры при запуске

	var a = new pageDraw({
		render: `
		<header>
			<h2></h2>
			<p>` + data + `</p>
		</header>
		<div>
			<ul>
				<li>1 блок</li>
				<li>1 блок</li>
				<li>1 блок</li>
				<li class="zzzz">1 блок</li>
			</ul>	
		</div>`,
		data: '/qqq/pageDrawTest.js',
		position: 'afterBegin body',
		element: 'section', // указывать только при инициализации
		class: 'qwe' // через пробел несколько классов
	});

a.render

Для того, что-бы записать любой html код внутрь блока нужно присвоить его внутри a.data,then(()=>{ a.render = code;}) если нужно получить данные или просто в любом месте если данные получать не нужно.



	a.data.then((data)=>{ 
		a.render = `
		<header>
			<h2></h2>
			<p>` + data + `</p>
		</header>
		<div>
			<ul>
				<li>1 блок</li>
				<li>1 блок</li>
				<li>1 блок</li>
				<li class="zzzz">1 блок</li>
			</ul>	
		</div>`;
	});

a.position

тут есть 3 варианта, или строка "afterbegin body" или массив, где 1 аргумент позиционирование, а 2 объект ['beforebegin',document.querySelector('body')] или массив срок, в случае если 1 вариант не подойдет то будет использоваться 2 вариант вставки.

a.children

если объекту 1 назнать объект 2 как ребенка, то объект 1 будет управлять отображением, перерендером и скрытием дочернего блока. Блоки могут находиться в любых местах, как вложенно так и нет. Если применяется пложенность, то иерархия должна соответствовать.

	var a = new pageDraw();
	var b = new pageDraw();

	a.children = b;

a.delete

Убирает блок из DOM и убирает из DOM детей
